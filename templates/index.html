Student Performance Prediction System

An End-to-End Machine Learning Application for Predicting Student Mathematics Performance

Project Overview

The Student Performance Prediction System is a machine learning-based web application designed to estimate a student's Mathematics score using demographic, educational, and academic attributes. The system utilizes historical student performance data and applies machine learning techniques to generate accurate predictions.

Users provide information such as gender, ethnicity, parental level of education, lunch type, test preparation status, reading score, and writing score. Based on these inputs, the trained machine learning model predicts the expected Mathematics score.

Machine Learning Pipeline

  • Data Ingestion and Data Validation
  • Feature Engineering and Data Preprocessing
  • Categorical Feature Encoding
  • Feature Scaling and Transformation
  • Model Training and Evaluation
  • Prediction Pipeline Deployment using Flask

Technology Stack

  • Python
  • Scikit-Learn
  • Flask
  • Pandas
  • NumPy
  • HTML & CSS
  • Machine Learning Pipelines

Objective

The primary objective of this project is to demonstrate the complete lifecycle of a machine learning application, from data preprocessing and model development to deployment and user interaction through a web interface.
